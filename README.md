# Cat-Has-Arrived

## Project Overview
"The Cat Has Arrived" is a serious game developed with Unity, designed to simulate the experience of a new cat owner. Through an isometric perspective, players engage in a cycle of Observation, Interaction, and Consequence to learn responsible pet care practices.
